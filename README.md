## 📊 How It Works

1. **Data Collection** - Fetches 10+ years of gold price data
2. **LSTM Training** - Neural network learns price patterns
3. **Prediction** - Generates future price forecasts
4. **Visualization** - Interactive charts show trends

## 🛠️ Tech Stack

- **Backend:** Flask, TensorFlow, Keras
- **Frontend:** HTML5, CSS3, JavaScript, Plotly
- **Data:** yfinance, pandas, numpy
- **Deployment:** Render/Railway
